Proiecte legate de programare (fiecare poză are link-ul către program)

Generator de parole

Acest program realizat în Python solicită ca date de intrare platforma (ex: Facebook, Twitter etc.) și numele de utilizator/email pe care îl doriți și oferă o parolă aleatorie. Parola este alcătuită din 20 de caractere alese aleatoriu din: toate literele alfabetului englez, caracterele zecimale (cifrele 1-10) și caracterele de punctuație. Realizat cu modulele Random și String.

Rezolvator de Sudoku

Acest program realizat în Python solicită ca date de intrare o listă de liste (matrice) ca reprezentant al unui tabel Sudoku (cu spațiile goale incluse). Programul rezolvă acest tabel Sudoku și umple toate spațiile goale.

Colectare de date de pe site-uri web

Acest program realizat în Python verifică în site-ul https://www.newegg.ca/ (un magazin de comerț electronic) un anumit produs dorit care este în stoc și afișează lista de produse sortate în ordine crescătoare în ceea ce privește prețul, pentru ca utilizatorul să îl aleagă pe cel mai ieftin. Exemplul se bazează pe un produs de tip GPU (placă video), dar programul funcționează pentru toate produsele disponibile pe acest site. Realizat cu modulele Beautiful Soup 4 și Requests.
